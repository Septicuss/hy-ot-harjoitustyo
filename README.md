## Installation

1. Install Poetry dependencies with

```
poetry install
```

2. Start the application
```
poetry run invoke start
```

## Commands

### Start the application
```
poetry run invoke start
```

### Testing
```
poetry run invoke test
```

### Coverage report
```
poetry run invoke coverage-report
```
